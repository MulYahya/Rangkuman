HTML

HTML (HyperText Markup Language) adalah bahasa markup yang digunakan untuk membuat halaman web dan menampilkan informasi di browser. HTML terdiri dari kode-kode tag yang menginstruksikan browser bagaimana menampilkan konten sesuai keinginan. Standar HTML saat ini dikelola oleh World Wide Web Consortium (W3C).

Sejarah HTML

Sejarah HTML dimulai dengan HTML 2.0 yang disetujui sebagai standar pada 22 September 1995. Kemudian HTML 3.2 hadir pada 14 Januari 1996, diikuti oleh HTML 4.0 pada 18 Desember 1997. HTML versi ISO/IEC 15445:2000 atau "ISO HTML" berdasarkan HTML 4.01 Strict dirilis pada 15 Mei 2000.
